# Diabetic-Retinopathy-Swin-Transformer
This project provides an end-to-end deep learning pipeline for classifying Diabetic Retinopathy (DR) severity using the APTOS 2019 Blindness Detection dataset. It automates the detection of retinal lesions to assist in early medical diagnosis.
